The GRaF project (2017-2019), short for [_Graz Repository of Ancient Fables_](gams.uni-graz.at/graf), was a citizen science project that aimed to build a repository and digital edition of ancient fables. It was conceived as both a scholarly and learner-facing digital edition. In this project, I was responsible for the technical implementation and developed the platform together with the project team. 
I also wrote much of the documentation  {% cite Lang2020Webportal, Lang2020Ressourcen, Lang2020Benutzung, Lang2020Dokumentation %}. 

We published range of publications about the digital humanities aspects {% cite graf2020zfdg, ianus2020graf, Lang2021GRaF, LangGartner2023GRaF %}.
