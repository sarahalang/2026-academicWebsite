[The GRaF project (2017-2019),](gams.uni-graz.at/graf) short for _Graz Repository of Ancient Fables_, was a citizen science project that aimed to build a repository and digital edition of ancient fables. It was conceived as both a scholarly and learner-facing digital edition. In this project, I was responsible for the technical implementation and developed the platform together with the project team. 
I also wrote much of the documentation on the web portal {% cite Lang2020Webportal %}, resources created within the project {% cite Lang2020Ressourcen %}, how to use {% cite Lang2020Benutzung %} and the technical documentation itself {% cite Lang2020Dokumentation %}. 

We published range of publications about the digital humanities aspects in an edited collection {% cite age2020 %} and in _ZfdG_ {% cite graf2020zfdg %} and about didactical uses of the portal {% cite ianus2020graf %}. The project was furthermore presented in the _KONDE Weißbuch_ {% cite Lang2021GRaF %}. and an edited collection on digital scholarly editions in Austria {% cite angGartner2023GRaF %}.
